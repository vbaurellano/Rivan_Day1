 - SR:		show run
 - SS:		show start
 - CRS:		copy run start
 - SPI:		show power inline
 - SIIB:	show ip interface brief
 - SVB:		show vlan brief
 - SIDB:	show ip dhcp binding
 - SMAC:	show mac address-table
 - SVPS:	show voice port summary
 - SDPVS:	show dial-peer voice summary
 - CS:		csim start [potsnum]
 
 *For Day 5
 
 - SES:		show etherchannel summary
 - SMT:		switchport mode trunk
 - STED:	switchport trunk encapsulation dot1q
 - STAV:	switchport trunk allowed vlan
 - STNV:	switchport trunk native vlan
